# ドキュメント

このディレクトリのドキュメントは [Diátaxis](https://diataxis.fr/) フレームワークに沿って
4つの目的別に整理している。読みたい目的に応じて入口を選ぶ。

| 目的 | 分類 | 入口 |
|------|------|------|
| **学びたい**（手を動かして覚える） | Tutorials | [tutorials/getting-started.md](tutorials/getting-started.md) |
| **作業したい**（特定の手順を知る） | How-to guides | [how-to/](how-to/) |
| **調べたい**（正確な情報を引く） | Reference | [reference/](reference/) |
| **理解したい**（背景・なぜを知る） | Explanation | [explanation/](explanation/) |

## 各分類の内容

### Tutorials（チュートリアル）

- [getting-started.md](tutorials/getting-started.md) — クローンしてから最初のプログラムをビルド・実行するまで

### How-to guides（ハウツー）

- [build-and-run.md](how-to/build-and-run.md) — 3つのビルド方法（CMake / 直接コンパイル / Dev Container）
- [add-new-project.md](how-to/add-new-project.md) — 新しいテーマ（プロジェクト）を追加する

### Reference（リファレンス）

- [cmake-and-structure.md](reference/cmake-and-structure.md) — ディレクトリ構成・CMake設定・ビルド生成物・ツール・命名規則

### Explanation（解説）

- [build-system-and-environment.md](explanation/build-system-and-environment.md) — なぜCMakeなのか・ビルドの仕組み・Dev Containerの仕組み・設計判断
