# Kuhn poker

研究用に作られた、世界一シンプルなポーカー。小さいのに「ブラフ」「コール/フォールドの
判断」というポーカーの本質が詰まっており、CFR の最初の題材として定番。

## ルール

- **2人対戦**
- **カードは3枚だけ**：J, Q, K（強さ J < Q < K）
- 両者 **1チップずつアンティ**（先に出す）→ ポットは 2
- 各自に **1枚配り**、1枚は伏せたまま
- **プレイヤー1（P1）から行動**。行動は **check（パス）** か **bet（1チップ賭ける）**

## 起こりうる展開（全5パターン）

```
P1 check → P2 check            → ショーダウン（強いカードがポットを取る）
P1 check → P2 bet → P1 fold    → P2 がポットを取る
P1 check → P2 bet → P1 call    → ショーダウン
P1 bet   → P2 fold             → P1 がポットを取る
P1 bet   → P2 call             → ショーダウン
```

## 利得（payoff）

- **fold した人はアンティ（1チップ）を失う**（相手が +1）。
- ショーダウンは**強いカードが勝つ**。
  - bet/call が無いショーダウン：勝者 +1（相手のアンティ）。
  - bet+call ありのショーダウン：勝者 +2（アンティ＋ベット）。

## 鍵となる概念：情報集合（information set）

ポーカーが難しいのは**相手のカードが見えない**こと（不完全情報）。プレイヤーは
「ゲームの完全な状態」ではなく「**自分が知っていること**」だけで判断する。この
「自分から見て区別がつかない状況のまとまり」を **情報集合** と呼ぶ。

Kuhn poker の情報集合 ＝ **(自分のカード) + (見えている行動履歴)**。

| プレイヤー | 情報集合の例 |
|-----------|-------------|
| P1 | 「J・最初の行動」「J・checkしたら相手がbet（call/fold判断）」… |
| P2 | 「Q・相手がcheck」「Q・相手がbet」… |

数えると **全部で12個**（各プレイヤー：3カード × 2つの判断ポイント = 6個ずつ）。

**戦略とは「各情報集合で、各行動をどの確率で選ぶか」**。例：「J を持って最初の行動
→ bet 20%, check 80%」。CFR が求めるのは、この **12個の情報集合それぞれの確率**。

## GTO（ナッシュ均衡）

**GTO ＝ どちらも戦略を変えても得をできない状態**（ナッシュ均衡）。Kuhn poker の重要事実：

- 最適戦略は**確率的（混合戦略）**。「常に bet」のような単純戦略は搾取される。
- **ブラフが最適戦略に現れる**：最弱の J でも一定確率で bet（ブラフ）するのが正解。
- **ゲームの価値は P1 にとって −1/18**（≈ −0.056）。完璧に打っても後手有利で P1 は
  1ハンドあたり平均 1/18 チップ負ける。これは**解析的に分かっている既知の答え**。

この「既知の答え」があるので、**自作の CFR が正しく収束したかを検証できる**
（[cfr.md](cfr.md) の検証を参照）。
